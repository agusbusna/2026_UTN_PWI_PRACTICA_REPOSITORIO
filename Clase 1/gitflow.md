## Gitflow
Con gitflow vamos a aprender a manejar un repositiorio colaborativo


NUNCA trabajamos sobre la rama main para agregar cambios, ya que estos pueden tener errores, fallos. para eso creamos una rama llamada: dev o development

Para crear la rama de desarrollo debemos usar 
    git checkout -b <nombre-nueva-rama>
Para movereme entre ramas 
    git checkout <nombre-rama>
